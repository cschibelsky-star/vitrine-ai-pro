# Validação do lote AiMemory

Este arquivo força a execução dos workflows de validação após a movimentação canônica de `AiMemory` para `App\Shared\AI\Models`.

Escopo validado:

- sintaxe PHP;
- bootstrap Laravel;
- inventário arquitetural;
- preservação do wrapper `App\Models\AiMemory`;
- ausência de alterações em migrations, tabelas e dados.
