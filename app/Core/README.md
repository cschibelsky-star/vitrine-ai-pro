# Core

Camada oficial de autenticação, usuários, empresas, produtos, planos, licenças, contratos, pagamentos, permissões, configurações e auditoria.

Regras:
- não depende de produtos específicos;
- não contém lógica de TV Digital, Guia Digital, SISMED, AssessorGov ou Compras IA;
- produtos consomem o Core por contratos e APIs estáveis.
