# Legacy

Área de quarentena para código antigo ainda não removido por depender de análise de impacto.

Regras:
- nenhum recurso novo entra nesta camada;
- mover somente após inventário de dependências;
- código legado não pode ser reutilizado como padrão de novos produtos;
- remoção exige teste de regressão e rollback disponível.
