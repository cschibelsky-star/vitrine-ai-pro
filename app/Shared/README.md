# Shared

Serviços técnicos reutilizáveis por mais de um produto: IA, OCR, PDF, armazenamento, upload, notificações, e-mail, WhatsApp, busca, importação e exportação.

Regras:
- não contém regra comercial específica;
- não conhece detalhes internos de um produto;
- expõe contratos estáveis e testáveis.
