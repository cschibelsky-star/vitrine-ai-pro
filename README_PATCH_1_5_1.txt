Master Start 1.5.1 — Valores automáticos em Contratos e Cobranças

Correções:
- Contratos / Propostas passam a preencher automaticamente valores ao selecionar o Plano.
- Mensalidade Base vem de plans.valor_mensal.
- Implantação vem de plans.valor_implantacao.
- Módulos Extras somam company_modules ativos de tipo extra/premium para o cliente/plano.
- Total Mensal Previsto = Mensalidade Base + Módulos Extras.
- Cobranças passam a preencher cliente, produto, plano, descrição e valor ao selecionar o contrato.

Observação:
Os campos continuam editáveis para negociação, proposta pública, desconto, cortesia ou implantação diferenciada.
